- Variável para guardar os valores somados (começa zerada)

- Loop que soma o valor de cada elemento a cada loop e depois divide pelo total de posições dos elementos dentro do array

1. Se o valor for 4 ou menor será exibido "Ruim"
2. Se o valor for 7 ou menor será exibido "Bom"
3. Se o valor for 10 ou menor será exibido "ótimo"
