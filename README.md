# Projeto_Calculadora_de_Medias_B7Web

Este projeto foi desenvolvido como parte de um desafio prático de lógica de programação do curso Fullstack da B7Web.

# Objetivos
O objetivo proposto pelo instrutor era analisar os requisitos de um sistema de notas e criar um passo a passo do algoritmo utilizando linguagem neutra (sendo opcional a tradução para uma linguagem de programação).

# Requisitos
- Receber uma lista de notas (de 0 a 10).
- Criar uma função para receber essa lista e calcular a média.
- Criar uma função para classificar a média obtida com base nas regras:
  - De 0 a 4 = Ruim
  - De 5 a 7 = Bom
  - De 8 a 10 = Ótimo
- Exibir os resultados na tela.

# Tecnologia
- JavaScript
